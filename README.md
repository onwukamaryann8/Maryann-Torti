DC 2025 Special Election, Structured JSON Dataset  
This dataset contains cleaned, structured, and machine-readable election data for the July 15, 2025 Ward 8 Special Election in Washington, D.C.  

It was generated from the official DC Board of Elections certified results file:  
'July_15_2025_Special_Election_Certified_Results.csv'

The dataset is organized into four JSON files, each representing an important part of the election information model used by our backend.
